Carga de datos

Puede utilizar los siguientes campos para ingresar productos, vendedores o registrar una venta

Registro de nuevo vendedor


*El campo de nombre es obligatorio
*El campo de nombre debe contener al menos {{contactForm.get('formularioVendedorNombre')?.errors?.['minlength'].requiredLength}} letras

*El campo de sueldo es obligatorio
*El campo de sueldo debe contener al menos {{contactForm.get('formularioVendedorSueldoBasico')?.errors?.['minlength'].requiredLength}} cifras

Registro de nuevo producto


*El campo de nombre es obligatorio
*El campo de nombre debe contener al menos {{contactFormProducto.get('formularioProductoNombre')?.errors?.['minlength'].requiredLength}} letras

*El campo de precio unitario es obligatorio
*El campo de precio unitario debe contener al menos {{contactFormProducto.get('formularioProductoPrecio')?.errors?.['minlength'].requiredLength}} cifras

*El campo de categoria es obligatorio
*El campo de categoria debe contener al menos {{contactFormProducto.get('formularioProductoCategoria')?.errors?.['minlength'].requiredLength}} cifras

Registro de nueva venta


*El campo de cantidad es obligatoria
*El campo de cantidad debe contener al menos {{contactFormVenta.get('formularioVentaCantidad')?.errors?.['minlength'].requiredLength}} letras

*El campo de monto total es obligatorio
*El campo de monto total debe contener al menos {{contactFormVenta.get('formularioVentaTotal')?.errors?.['minlength'].requiredLength}} cifras

*El campo de codigo de producto es obligatorio
*El campo de codigo de producto debe contener al menos {{contactFormVenta.get('formularioVentaProducto')?.errors?.['minlength'].requiredLength}} cifras

*El campo de codigo de vendedor es obligatorio
*El campo de codigo de vendedor debe contener al menos {{contactFormVenta.get('formularioVentaVendedor')?.errors?.['minlength'].requiredLength}} cifras
